# 阶段 3.5: 人味化改写

这是整个 skill 最重要的一层人工质感控制。

## 必过清单

1. **句长抖动**:长短句混用，不要连续四句一个长度
2. **禁用套话**:少用“首先/其次/综上所述/随着...的发展”等
3. **AI 高频词**:少用“赋能/闭环/抓手/底层逻辑/一站式/革命性”等
4. **开头不要宏观背景**:优先具体场景、数字、原话、日志、问题
5. **加入第一人称和立场**:允许不确定，允许主观判断
6. **提高事实密度**:每 500 字内至少出现具体数字、时间、人名、产品名或版本号
7. **标点更自然**:允许 `——`、问号、括号插入，不要只有逗号句号
8. **允许轻微不完美**:小节长短不对称、回收一句、自嘲一句都可以
9. **最后按账号 voice 过一遍**:只使用当前账号 `voice`，不要引用文档里的旧账号画像

## 常见禁用表达

```text
首先 / 其次 / 最后 / 值得一提的是 / 不可否认 / 毋庸置疑 /
综上所述 / 总而言之 / 由此可见 / 众所周知 /
在...的背景下 / 随着...的发展 / 站在...的角度
```

## 常见 AI 高频词

```text
赋能 / 打造 / 聚焦 / 生态 / 闭环 / 抓手 / 方法论 / 底层逻辑 /
提升效率 / 助力 / 一站式 / 端到端 / 核心竞争力 / 破局 / 沉淀
```

命中这些词时，优先重写整句，不只是替换词。
